# Changelog

All notable changes to the Quarto Theme Tweaker extension are documented here.

## [0.0.3] - 2026-06-25

- Command palette entry now uses a "Theme Tweaker" category instead of a "Quarto:" prefix, so it no longer looks like it ships with the official Quarto extension.
- Added a user-facing Usage section to the top of the README.

## [0.0.1] - 2026-06-20

- Initial release.
- Live Sass theming for Quarto documents: sliders and color pickers bound to Sass variables.
- Instant preview with write-back to `custom.scss`.
- Presets for `quarto-html` and `quarto-revealjs`.
